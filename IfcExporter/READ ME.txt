## To build the project you need to open Visual studio in Administration mode then follow these steps:

1) Clean the solution.
2) Rebuild IfcExporter project.
3) Open the Developer PowerShell then use these commands:
	1- cd "<Path to IfcExporter project>\obj\Debug"
	2- gacutil /i IfcExporter.dll
	3- C:\Windows\Microsoft.NET\Framework64\v4.0.30319\regasm IfcExporter.dll /tlb:com.IfcExporter.tlb /codebase

4) Rebuild the solution
